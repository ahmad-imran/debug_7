The main difference between localStorage and sessionStorage is that sessionStorage is unique per tab. If you close the tab the sessionStorage gets deleted, localStorage does not. Another difference is localStorage has a limit of 2551 k characters but sessionStorage has unlimited storage.

Block Scope: The block scope of a variable means that the variable is accessible within the block that is between the curly braces. You can't access them from outside that block.

Global Scope: Variables declared Globally (outside any function) have Global Scope. Global variables can be accessed from anywhere in a JavaScript program. Variables declared with var, let and const are quite similar when declared outside a block.

An event loop is an infinite loop that waits for tasks, executes them, and then sleeps until it gets additional tasks. JavaScript's runtime model is built on an event loop, which is in responsible of running code, collecting and processing events, and performing queued sub-tasks. The event loop is a technology that allows JavaScript to conduct non-blocking activities. When an asynchronous job is launched, the event loop begins to execute. When the job is finished, the event loop will check for any remaining tasks.

Undefined means a variable has been declared but has not yet been assigned a value. undefined is a type by itself (undefined).
  • A variable that has not been assigned a value is of type undefined.
  • A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value.
  • A function returns undefined if a value was not returned.
  • Accessing a non-existing property
  • Without return statement, a JavaScript function returns undefined
  • You get undefined when accessing an array element with an out of bounds index.
